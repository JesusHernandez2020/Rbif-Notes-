# bash scripting

## How to run the script 

--> the file type is .sh

--> At the start of the script `#!/bin/bash`

--> to run the script, run the cmd: `bash` script.sh or use `chmod +x my_script.sh` to make it an executable. now you can run it as ./script.sh
